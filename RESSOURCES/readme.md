# Projet-MIAGE-E7-Gestionnaire-Fiches-

Application permettant de gérer des fiches de personnages de type Jeu de Rôle. Un utilisateur peut créer un compte protégé par un mot de passe, à partir duquel il peut créer des fiches de personnages. Chaque fiche peut être modifiée de manière modulaire par l’utilisateur.


# Fonctionnalités principales
Les fonctionnalités suivantes doivent êtres toutes implémentées.
    - comptes protégés par mot de passe ;
    - sauvegarde des comptes et des fiches crées ;
    - chaque fiche possède les éléments (modules) suivants :
    - portait , biographie, statistiques, compétences, équipement,
      (et tout autre éléments jugés utiles);
    - ces éléments doivent être modifiables ; par exemple : on peut
      changer le nom d’une statistique, ajouter des compétences, etc.

# Fonctionnalités secondaires
Les fonctionnalités suivantes sont facultatives et indépendantes les unes
des autres.
    - cryptage des fichiers sauvegardés ;
    - possibilité de modifier le positionnement des modules sur la fiche ;
    - possibilité de créer et importer des templates;
